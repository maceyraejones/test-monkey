## Deploy Locally

1. Go to VSCode and go to Terminal

2. Clone Repo (using Git Bash)

` git clone https://github.com/maceyraejones/test-monkey/ ` 

3. Install [Node](https://nodejs.org/en/download/)

4. Run App (localhost:3000)

` npm run dev `
